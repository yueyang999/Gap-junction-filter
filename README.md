# Gap-junction-filter
# [Retinal Gap Junction Inspired Neural Networks Facilitate Blind Denoising and Classification in Visual Hierarchy](https://www.biorxiv.org/content/10.1101/2022.05.16.491952v1)

This repository contains data and code from the paper [Retinal Gap Junction Inspired Neural Networks Facilitate Blind Denoising and Classification in Visual Hierarchy](https://www.biorxiv.org/content/10.1101/2022.05.16.491952v1) by Yang Yue, Kehuan Lun, Liuyuan He, Gan He, Shenjian Zhang, Lei Ma, Jian K Liu, Yonghong Tian, Kai Du, and Tiejun Huang. 

## data
This subdirectory contains all images and statictics used for generating figures in our experiments.

## DCNNs
This subdirectory contains code for blind denoising, integrated gradient analysis and classification in our experiments. The training and testing dataset for blind denoising come from the paper [Beyond a Gaussian Denoiser: Residual Learning of Deep CNN for Image Denoising](https://arxiv.org/abs/1608.03981). The training and testing dataset for classification come 
from the paper [Generalisation in humans and deep neural networks](https://arxiv.org/abs/1808.08750).

## detailed photoreceptor network
This subdirectory contains code implemented by NEURON 7.4 for detailed photoreceptor model, its network and STA analysis.

## gap junction filter
This subdirectory contains code implemented by NEURON 7.4 for simplified photoreceptor network, i.e. G-filter.

## figures
Experiment results in our paper are generated by the following files:

`draw_fig1.m`: Draw the results of STA analysis on the detailed photoreceptor network.

`draw_fig2.m`: Draw the denoising performance histograms of all filters.

`draw_fig3_No34_noise_hist.m`: Draw the noise histogram for image No.34 in BSD68.

`draw_fig3_noise_average_distribution.m`: Draw the average noise distributions for all images in BSD68.

`draw_fig4_mixed.m`: Draw the denoising performance curves, where the testing dataset contains mixed noise.

`draw_fig4_SP.m`: Draw the denoising performance curves, where the testing dataset contains Salt-and-Pepper noise.

`draw_fig5_sample_map.m`: Draw several attribution maps of image No.34 in BSD68.

`draw_fig5_statistics.m`: Draw the statistics of all attribution maps, including distributions and boxplot.

`draw_fig6.m`: Draw the classification performance curves.
